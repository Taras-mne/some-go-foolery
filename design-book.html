03

Components

Minimal, functional components with sharp edges (2px radius). No shadows, no gradients. Borders are thin (1px) and subtle. The red accent is used sparingly for primary actions and active states.

Buttons

Primary
Main actions. Red background, white text. Use sparingly — one per section.
Secondary
Secondary actions. Transparent with border. Most common button type.
Ghost
Tertiary actions. Muted color, subtle border. For low-emphasis actions.

Animations

Subtle, functional animations. No decorative motion. All animations are short (0.15s-0.4s) with simple easing. The only continuous animation is the status indicator blink.

Logo

The logo is a pixel-art cloud made of white rectangles on a red square. The wordmark is always in Geist Mono, lowercase. The red square can be used as a standalone icon.
